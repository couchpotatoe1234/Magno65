## Magno65
After weeks of hard work, here it is. The first prototype of the Magno65.
<img width="1332" height="605" alt="Screenshot 2026-04-17 at 3 54 28 PM" src="https://github.com/user-attachments/assets/01d262dc-8011-43ef-abcc-6425135e1d1e" />
It will use N52 Neodymium magnets that are 5mmx2mm in the PCB and on the bottom of the case, allowing the plate and the PCB to levitate. The case itself is two parts, so it is 3D-printable. I currently have some untested firmware that will be tested when I make this project. Heres the aproximate BOM currently.
